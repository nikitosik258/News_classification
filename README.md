# NLP Classification: Data Science Job Classification

## Задача
Классификация вакансий на Data Science и не-Data Science позиции на основе анализа текстового описания и заголовка вакансии.

## Данные
- **Источник:** Описания вакансий с различных job-сайтов. Можно скачать по ссылке: https://disk.yandex.ru/d/l068c8NnHpY02w
- **Размер:** 1,624 записи
- **Признаки:**
  - `title` - заголовок вакансии
  - `description` - подробное описание позиции
- **Целевая переменная:** `is_data_science` (0/1)
- **Особенности данных:**
  - Описания в markdown формате
  - Содержат технические термины и требования
  - Сбалансированный датасет

## Решение

### 1. EDA (Исследовательский анализ данных)
- Анализ структуры и качества данных
- Исследование распределения длины текстов
- Анализ важности токенов для классификации
- Выявление ключевых слов и фраз

### 2. Baseline модели
- **CountVectorizer + классические ML алгоритмы:**
  - LogisticRegression
  - DecisionTreeClassifier  
  - RandomForestClassifier
  - LGBMClassifier
- **TF-IDF векторизация**
- **Очистка данных:** удаление стоп-слов, сохранение пунктуации

### 3. Нейронные сети
- **RNN** - рекуррентная сеть для анализа последовательностей
- **CNN** - сверточная сеть для выявления локальных паттернов
- **LSTM** - долгосрочная память для сложных зависимостей
- **CNN + GloVe** - предобученные эмбеддинги слов

### 4. Предобработка
- Токенизация с помощью tiktoken
- Создание последовательностей для нейронных сетей
- Отбор важных токенов
- Стратифицированная кросс-валидация
